Car Parking Sensor

Computer Programming Project

KMITL Semester 2/2021 1st Year

Objective

โปรเจคนี้มีจุดประสงค์เพื่อ ความปลอดภัยของการถอยรถ เราจึงสร้างระบบกันชน โดยการพัฒนาระบบกันชนรถ เราได้ใช้อุปกรณ์ไมโครคอนโทรเลอร์ Arduinoและ hc-sr04 ultrasonic sensor ตรวจจับ ซึ่งจะทำให้ผู้ขับขี่รถถอยรถอย่างปลอดภัย

Origin

ปัจจุบันจะเห็นได้ว่าโดยส่วนใหญ่ รถรุ่นใหม่ๆได้มีการติดตั้งระบบกันชนแล้ว แต่ตอนนี้ยังมีผู้ขับขี่รถรุ่นเก่าที่ยังไม่มีระบบกันชน ทางกลุ่มได้เล็งเห็นว่าทุกคนควรได้รับความปลอดภัยในด้านนี้ จึงพัฒนาระบบกันชนขึ้นมา เพื่อตอบโจทย์ ผู้ขับขี่รถรุ่นเก่าที่ยังไม่มีระบบกันชน

Tools

Ardino Uno R3

Hc-sr04 ultrasonic sensor

Ssd1036 oled display

Buzzer

Resistor 1k

Male to Male Jumper
